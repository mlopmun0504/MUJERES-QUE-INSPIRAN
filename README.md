# MUJERES-QUE-INSPIRAN
"Mujeres que inspiran: Un viaje deportivo por Andalucía" es una Situación de Aprendizaje (SdA) interdisciplinar (Educación Física, Lengua y Conocimiento del Medio) diseñada para 5º de Primaria.  El alumnado, organizado en equipos provinciales, investiga y visibiliza a referentes femeninos del deporte andaluz.
